# 02 ÔÇö `ajolote init`

Tests for the `ajolote init` command: scaffolding the `.agents/` directory, importing existing tool configs, deduplicating commands, and setting up `.gitignore`.

**Start each test** in a fresh directory: `mkdir ~/test-init && cd ~/test-init && git init`

---

### INIT-001 ÔÇö Fresh project, no existing tool configs

**Prerequisites:** Empty git repo; no `.agents/`, no `.claude/`, no `.cursor/`.  
**Steps:**
1. `ajolote init`
2. `ls .agents/`
3. `ls .agents/rules/ .agents/skills/ .agents/personas/ .agents/context/ .agents/commands/`
4. `cat .agents/config.json`

**Expected result:** `.agents/config.json` created. Starter markdown files present under `rules/`, `skills/`, `personas/`, `context/`, and `commands/`. Config JSON contains `rules`, `skills`, `personas`, `context` arrays pointing at those files. Output prints "Initialized" message.  
**Pass / Fail:** ÔİÉ
